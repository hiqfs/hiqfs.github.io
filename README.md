-------------
### Help
开发环境：<br />
请事先安装Node.js软件包 执行命令npm install<br />
安装完后执行npm test命令，就可以访问http://127.0.0.1/ 开始测试了<br />
文件结构：<br />
src是源代码。static是为了提高访问速度由源代码生成的压缩静态文件。<br />
web是压缩js和css脚本，auto是自动提交git项目shell脚本<br />
